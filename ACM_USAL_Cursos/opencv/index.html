Visión Artificial con OpenCV

La visión artificial es el conjunto de técnicas que permite a un ordenador comprender las características de una imagen, reconociendo objetos, transformando la imagen en una nueva, o incluso crear realidades aumentadas.

En este seminario realizaremos una introducción a la biblioteca abierta OpenCV, que incluye un gran conjunto de herramientas para el procesado y reconocimiento de imágenes, tanto para ordenadores como equipos móviles (smartphones, tablets...).

El seminario forma parte de los cursos de formación permanente de la USAL por lo que todos los asistentes recibirán un certificado acreditativo.

¿Qué necesito?

OpenCV para C/C++

Realizaremos varios ejercicios con la API de OpenCV para C/C++, por lo que será necesario que antes de venir al taller instales el software. Existen versiones para Windows, OS X y GNU/Linux, integrables con IDEs como Visual Studio, XCode o Eclipse.

Trabajaremos principalmente con C++, por lo que es conveniente tener conocimientos básicos de la Programación Orientada a Objetos.

  • Instalación en Windows con Visual Studio
  • Instalación en OS X con XCode
  • Instalación en GNU/Linux, e integración con Eclipse

Intro

Se verá una introducción a lo que es el sistema Git, creado por Linus Torvalds, desde un punto de vista general.

Posteriormente se hablará de la imporancia del sistema en la ingeniería informática en la actualidad, y como nos puede sacar de un serio apuro en nuestra vida profesional.

Se repasarán algunas de las características técnicas para finalizar la introducción del taller.

Coding

Se verán los comandos mas interesantes, populares y útiles de Git si vamos a utilizar el sistema desde la línea de comandos. Se hará de manera pausada y comprensiva, explicando cada uno de los comandos, y permitiendo a los asistentes poder seguir el taller de una manera cómoda y entretenida.

También se dará un vistazo a la posibilidad de uso de git con clientes de escritorio, aunque esto sea de n00bs ;)
