Segundo semestre

A partir de marzo, tenemos preparados más de 10 seminarios en la Facultad cubriendo áreas tan diversas como el desarrollo de videojuegos, la realidad virtual, gestión de código o simulación de procesos naturales. ¿Te apuntas?

Todos los cursos son gratuitos, y cuentan con el reconocimiento del Centro de Formación Permanente de la Universidad de Salamanca.

Python y Biotecnología

¿Quieres conocer las aplicaciones que tiene la programación en el mundo de la biología? En este seminario presentaremos el lenguaje de programación Python, su filosofía y su potencial así como sus fundamentos básicos y funciones principales. Después repasaremos conceptos biológicos básicos y propondremos una serie de problemas a resolver mediante el diseño e implementación de algoritmos con Python.

Fecha: 19 de marzo

Git + GitHub

¿Estás cansado de tener que crear varias versiones de cada proyecto o práctica, o de perder código por no hacer una copia de seguridad? ¿Quieres conocer una forma efectiva de compartir código con otros programadores? En este seminario daremos una introducción a los conceptos básicos de git, y utilizaremos GitHub para alojar nuestro código y compartirlo con otros programadores, así como las herramientas que proporciona este servicio para gestionar equipos y el código alojado

Processing

¿Cómo podemos representar las propiedades de fenómenos naturales en continuo cambio mediante software? ¿Cómo entender los principios matemáticos que rigen los fenómenos físicos para crear mundos digitales? En este semniario nos centraremos en estrategias de programación y técnicas de simulación de procesos naturales utilizando la herramienta Processing.

FIWARE

FIWARE es una plataforma basada en la nube para la creación y distribución de servicios y aplicaciones innovadoras a coste reducido. En esta charla hablaremos de cómo utilizar los recursos proporcionados por FIWARE para crear aplicaciones web y que aprovechen el Internet of Things.

Unity

El motor de desarrollo Unity3D es una herramienta sencilla para crear juegos y aplicaciones 3D que funcionan en más de 20 plataformas distintas. En este seminario daremos a conocer los conceptos básicos del motor y cómo utilizarlo con varios ejemplos prácticos, diseñando una escena básica. Además, sirve de base para el taller con Google Cardboard.

OpenCV

La biblioteca OpenCV permite crear software capaz de reconocer qué ve, generar entornos de realidad aumentada o procesar imágenes y vídeo en tiempo real. En este seminario enseñaremos cómo utilizar la biblioteca e integrarla con el resto de componentes de nuestra aplicación.
